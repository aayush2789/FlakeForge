Liveness for Docker and Space health checks.
POST /reset
Start an episode. JSON body may include test_identifier, repo_path, max_steps, num_runs. Returns an initial observation.
POST /step
One environment step: agent fields such as think_text, patch_text, and predictions. Returns observation, reward, done.
